And now...it get's very exciting! We move into third dimension!
              
Along with `data-x` and `data-y`, you can define the position on third (Z) axis, with `data-z`.


<div id="big" class="step" data-x="3500" data-y="2100" data-rotate="180" data-scale="6"a>

            
           
            
CSS Transformation Syntax

            
The element should be positioned far away from us (by 3000px).

    
css(el, {
  position: "absolute",
  transform: "translate(-50%,-50%)" +
             translate(step.translate) +
             rotate(step.rotate) +
             scale(step.scale),
  transformStyle: "preserve-3d"
  });
};
