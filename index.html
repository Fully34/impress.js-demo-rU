Welcome to impress.js! This is the first slide of this presentation framework. Each step of the presentation should be an element inside of a div with an id name of 'impress' and a class name of 'step'. These 'step' elements are positioned, rotated and scaled by impress.js, and the 'camera' shows them on each step of the presentation.
Positioning information is passed through data attributes. The first three steps of the presentation are not scaled or rotated. Therefore, we only specify an x and y position of these step elements. The first step was positioned with 'data-x="-1000"' and 'data-y="-1500"' attributes. This means that the center of the element (yes, the center) will be positioned in point x = -1000px and y = -1500px of the presentation 'canvas'.
The 'id' attribute of the step element is used to identify it in the URL, but it is optional. If it is not defined, each step will get a default value of 'step-N' where N is a number of slide. The hash part of the URL when this step is active will be '#/step-2'. You can also use '#step-2' in a link, to point directly to this particular step.
This is a scaled element dictated by the data attribute 'data-scale="4"'. This means that this element is 4 times larger than the others.
